About

Lithium is a minimal Linux distro compiled with the latest kernel, busybox, and a custom package manager. Perfect for tinkering, learning, and small systems.

Features

  • Latest Linux kernel
  • Minimal userspace
  • Custom package manager (LPM)
  • Network-ready
  • Optimized ISO under 17MB
