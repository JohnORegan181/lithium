how to connect to the internet (eithernet)

  1. set eth0 (eithernet) up: ifconfig eth0 up
  2. now find a ip address you can use with udhcpc (make sure to remember/write down the two ip's you see at the bottom): udhcpc -i eth0
  3. now lets set the ip address: ifconfig eth0 [the leftmost ip you got from uchcpc] netmask 255.255.255.0
  4. now set the default gateway : route add default gw [rightmost ip you got from udhcpc]
  5. now you're connected to the internet test it by running: lpm lithfetch
  6. to run lithfetch just run: lithfetch

how to update the list file so you can download more packages

  1. change your working directory to root: cd /
    2. clear the list: rm -rf list
  2. run: wget https://raw.githubusercontent.com/JohnORegan181/lithium/refs/heads/main/boot-files/initramfs/list
  3. Enjoy your updated package list!
